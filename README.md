# PlayWithDWZ
PlayFrameWork 1.27 与 国产UI 框架的 DWZ 的整合Demo。


PlayFrameWork 为后端

DWZ 为前端

数据交互全用 Ajax

###运行方式
安装Paly Framework，在代码根目录执行命令：

    play run

运行成功后访问: http://localhost:9000/ 即可.
